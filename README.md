Show hidden files : cmd + shift + . (period)


```
$ zsh
$ source ~/.zshrc
```

Open in Finder
```
$ open .
```

Clear Terminal
```
$ clear
```
or cmd + k

List Files in directory
```
ls -la
```

Current running process
```
echo $0
```

Current shell
```
echo $SHELL
```


Change shell
```
$ chsh -s /bin/zsh
```

nano move line
- Ctrl + k (to cut a line)
- Ctrl + u (to paste)

```
$ which ls
```

Use original command
```
$ command ls
```

```
$ echo $USERNAME
$ echo $HOST
$ echo $PS1
$ echo $PROMPT
```
